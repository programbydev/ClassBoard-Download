# DGClassBoard Changelog

A summary of what changed in each public release. For each version, download the
installer from the [latest release page](https://github.com/programbydev/ClassBoard-Download/releases/latest).

## v1.1.0

### New

- **What's New screen.** Open Help > What's New to see the highlights of each
  update. It also appears automatically the first time you launch a new version.

### Improved

- **Professional 3D workspace.** 3D objects now use a realistic lighting setup so
  that matte, gloss, metal, and glass materials look clearly different, and curved
  shapes such as spheres and cylinders shade smoothly instead of looking flat.
  (3D is a Pro feature.)

### Fixed

- Deleting or duplicating a page or notebook no longer switches you to a
  different page or notebook by mistake.
- Removing a section that contained all of a notebook's pages no longer leaves
  the notebook empty.
- Reset to Defaults now updates every setting on screen immediately.
- Page tags can now be removed no matter how they were capitalized.
- Duplicating a page now respects the page limit during a restricted session.

## v1.0.0

- First public release of DGClassBoard for Windows.
- Digital whiteboard with notebooks, sections, and pages.
- Pen, shape, connector, text, image, table, and flowchart tools.
- Export to PNG, PDF, and SVG, and save/reopen `.classboard` files.
- 30-day Trial with Pro activation via developer-generated license keys.
- Light, Dark, Graphite, and System themes with Microsoft Fluent UI icons.
