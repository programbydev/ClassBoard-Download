## What You Can Do With DGClassBoard

- Create digital whiteboard pages
- Write and draw with pen tools
- Use different brush styles
- Add shapes and diagrams
- Create flowcharts
- Connect objects with connector lines
- Add text blocks and text on a curve
- Insert images
- Create tables
- Import PDF pages
- Export work as PNG, PDF, or SVG
- Save and reopen `.classboard` files
- Submit feature requests, bug reports, or Pro purchase requests from Help > Request Form
- Use presentation-style tools such as laser, spotlight, and curtain overlays


## Main Features

- Notebook, section, and page based organization
- Infinite canvas style workspace
- Pen, eraser, selection, hand, ruler, crop, shape, connector, and text tools
- Microsoft Fluent UI System Icons with theme-aware contrast
- Light, Dark, Graphite, and System theme modes
- Flowchart symbols for process diagrams
- Rich text formatting
- Image editing options such as resize, crop, border, shadow, opacity, and flip
- Table editing with rows, columns, cell selection, and merge support
- Check for Updates support
- 30-day Trial with standard features enabled
- Pro activation through developer-generated license keys
- Help > Request Form for New Feature, Bug Report, and Purchase Pro Version requests

Google Drive integration and 3D functionality are Pro-only features.

## Privacy

For normal offline whiteboard use, DGClassBoard does not require a login and does not upload your personal identity.

Your whiteboard files stay on your computer unless you choose to save, export, share, or upload them yourself.

Optional features may use limited information only when required:

- License activation may use license/device information to activate Pro access.
- Google Drive sync works only if you choose to sign in with Google and have Pro access.
- Update checking only checks the current app version against the public update metadata.
- Help > Request Form asks for your name, email address, and request details. Requests are saved locally and may be sent to the developer only when a secure support endpoint is configured. Attachment file contents are not uploaded by the form.

Do not share your license key publicly.
