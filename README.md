# ClassBoard

ClassBoard is a Windows digital whiteboard application for teaching,
presentations, notes, diagrams, and classroom-style explanation.

Download the latest installer:

https://github.com/programbydev/ClassBoard-Download/releases/latest


## Download

1. Open the latest release page.
2. Download the `.exe` installer file.
3. Run the installer on Windows.
4. Launch ClassBoard from the Start Menu or desktop shortcut.

Current public version: **v1.0.0**


## What You Can Do With ClassBoard

- Create digital whiteboard pages
- Write and draw with pen tools
- Use different brush styles
- Add shapes and diagrams
- Create flowcharts
- Connect objects with connector lines
- Add text blocks
- Insert images
- Create tables
- Import PDF pages
- Export work as PNG, PDF, or SVG
- Save and reopen ClassBoard files
- Use presentation-style tools such as laser, spotlight, and curtain overlays


## Main Features

- Notebook, section, and page based organization
- Infinite canvas style workspace
- Pen, eraser, selection, hand, ruler, crop, shape, connector, and text tools
- Flowchart symbols for process diagrams
- Rich text formatting
- Image editing options such as resize, crop, border, shadow, opacity, and flip
- Table editing with rows, columns, cell selection, and merge support
- Google Drive support when the user chooses to sign in
- Check for Updates support
- Trial and Pro activation support


## Built With

ClassBoard is built using:

- C++17
- Qt 6
- QML
- CMake
- Inno Setup for the Windows installer


## Privacy

For normal offline whiteboard use, ClassBoard does not require a login and does
not upload your personal identity.

Your whiteboard files stay on your computer unless you choose to save, export,
share, or upload them yourself.

Optional features may use limited information only when required:

- License activation may use license/device information to activate Pro access.
- Google Drive sync works only if you choose to sign in with Google.
- Update checking only checks the current app version against the public update
  metadata.

Do not share your license key publicly.
